# 💄 AI美妆私教

基于腾讯云人脸识别API的智能妆容推荐应用，通过分析用户面部特征（脸型、眉型、眼型、鼻型、嘴型、肤质等），结合不同场景需求，自动生成个性化妆容建议并推荐相关视频教程。

## 项目简介

本项目是一款跨平台移动应用，帮助用户了解自己的面部特征并获取科学、个性化的妆容建议。用户只需上传一张正面自拍照，选择使用场景（通勤、约会、面试、上学），系统即可智能分析面部特征并提供针对性的妆容指导。

## 技术栈

| 技术领域 | 选型 |

## 快速开始

### 1. 克隆项目

git clone <仓库地址>
cd ai-makeup-coach

### 2. 配置后端

进入后端目录，安装依赖并配置密钥：

cd beauty-server
npm install

在 beauty-server 目录下创建 .env 文件，填入你的腾讯云密钥：

SECRET_ID=你的SecretId
SECRET_KEY=你的SecretKey

### 3. 启动后端

node server.js

启动成功后会显示：

🚀 后端服务已启动: http://localhost:3000

### 4. 运行前端

使用 HBuilderX 打开项目，点击“运行”→“运行到浏览器”或“运行到手机/模拟器”。

注意：如果使用手机预览，需要将前端 index.vue 中的 apiBaseUrl 从 http://localhost:3000 改为电脑的局域网 IP（如 http://192.168.x.x:3000），且手机需与电脑连接同一 WiFi。

## 功能说明

| 功能 | 描述 |
| ---- | ---- |
| 场景选择 | 支持通勤、约会、面试、上学四种场景 |
| 照片上传 | 支持拍照或从相册选择正面照片 |

## 密钥获取

本项目使用腾讯云人脸识别服务，需要先开通并获取密钥：

1. 登录 腾讯云控制台
2. 搜索“人脸识别”并开通服务
3. 进入 API密钥管理 获取 SecretId 和 SecretKey
4. 将密钥填入 .env 文件

注意：腾讯云人脸识别每月提供 1 万次免费调用额度，足以满足开发测试需求。

## 注意事项

- 上传的照片需为正面、清晰、无遮挡，避免美颜和滤镜
- 手机预览时需与电脑连接同一 WiFi
- 密钥仅存放于 .env 文件中，切勿提交到 GitHub
